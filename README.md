# **Fitur Utama**

1. Konversi Suhu Antar Skala
  a. Dari Celsius ke Fahrenheit, Kelvin, dan Reamur.
  b. Dari Fahrenheit ke Celsius, Kelvin, dan Reamur.
  c. Dari Kelvin ke Celsius, Fahrenheit, dan Reamur.
  d. Dari Reamur ke Celsius, Fahrenheit, dan Kelvin.

2. Validasi Input
  a. Menangani error jika pengguna memasukkan nilai bukan angka (ValueError).
  b. Memberikan pesan khusus jika suhu Kelvin dimasukkan negatif (karena tidak valid).

3. Fitur Tambahan Thermometer Skala 1–10
  a. Menampilkan intensitas panas dalam bentuk bar visual.
  b. Skala 0–10 berdasarkan suhu dalam Celsius.
  c. Memberikan status deskriptif:
    1) 0 → Sangat Dingin
    2) 1–3 → Normal
    3) 4–6 → Hangat
    4) 7–9 → Panas
    5) 10 → Ekstrem

4. Menu Interaktif
  a. Pengguna dapat memilih satuan suhu awal (Celsius, Fahrenheit, Kelvin, Reamur).
  b. Opsi keluar dari program.
  c. Program berjalan dalam loop sehingga bisa digunakan berulang kali.

5. Fitur Lanjut/Tidak Lanjut
  a. Setelah konversi, pengguna ditanya apakah ingin melanjutkan.
  b. Jika memilih tidak, program berhenti dengan pesan terima kasih.
